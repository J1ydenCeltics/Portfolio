# Portfolio

An interactive, responsive web portfolio showcasing my journey as an aspiring **Cybersecurity Analyst**. This project serves as a central hub for my technical skills, academic goals, and functional coding projects.

## 🚀 Live Demo Features
Unlike static portfolios, this site includes a built-in **Project Sandbox** using a custom modal system. 

### Interactive Tools & Games:
* **Python Calculator:** A functional math tool using JavaScript logic.
* **Tic-Tac-Toe:** A logic-based game demonstrating state management.
* **Password Generator:** A security-focused tool for creating 12-character complex strings.
* **Security Quiz:** A knowledge-check tool focused on programming and security best practices.
* **Mini Spreadsheet & Profile Maker:** Practical examples of data handling and DOM manipulation.

## 🛠️ Technical Stack
* **Frontend:** Semantic HTML5, CSS3 (Modern Dark Theme/Neon Accents)
* **Logic:** Vanilla JavaScript (ES6+)
* **Certification Focus:** CompTIA Security+ prep

## 🛡️ Skills Focus
The "Skills" section is designed to be interactive. Clicking on a skill tag provides a brief, human-centric explanation of how I apply that specific technology in my coursework or personal labs. 

Highlights include:
* **Linux & Command Line:** System administration and environment management.
* **Security Tools:** Network scanning and environment securing.
* **Automation:** Using Python to create custom tools for technical efficiency.
